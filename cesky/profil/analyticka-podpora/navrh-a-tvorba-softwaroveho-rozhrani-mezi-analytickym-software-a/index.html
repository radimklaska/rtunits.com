Velmi důležitou částí pro zapojení on-line analyzátorů do komplexního řídícího procesu je tvorba rozhraní mezi řídicím systémem a analytickým software, který se stará o zpracování a vyhodnocení jednotlivých spekter nebo analýz.

Pro naše aplikace používáme kombinace produktů Controlweb® www.controlweb.cz a analytického software Clarity. www.dataapex.com.

Naši specialisté pro tvorbu kontrolních systémů vytvoří požadovaný software pro komunikaci mezi jednotlivými programy. Data získaná analýzami procesních proudů pak mohou být použita pro okamžitou procesní regulaci.
